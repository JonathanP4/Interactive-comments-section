## Overview

### Built with:

* TypeScript
* React
* Redux/React-redux
* Redux Toolkit
* Vite (bundler)
* CSS Modules

### Features:

* Edit your replies and/or comments
* Reply to a comment or another reply
* Delete your comments
* Comments/replies are stored in local storage
